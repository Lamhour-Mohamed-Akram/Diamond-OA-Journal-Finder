Open Access Journal Finder

Find open access journals — Diamond (100% free to publish) by default, or include APC journals with one click. Everything runs right here in your browser (nothing is uploaded anywhere) and the result is remembered for next time.

One click — uses the DOAJ + SCImago snapshots shipped with the app (a few MB, then cached on this device). Want the very latest data instead? Follow steps 1–2 below.
— or load fresh files yourself —

⬇ Download the DOAJ journal CSV
The download starts by itself when you open the link (~25 MB, all open access journals). Then drop the file in the box below.
2Rankings (SCImago)waiting…
↗ Open scimagojr.com/journalrank.php
On that page, click the “⬇ Download” button at the top right. It saves a file like “scimagojr 2025.csv” — then drop that file in the box below.
💡 Tip: for the best results, download the full default list (no filters) — you can filter inside this app afterwards. Filtered downloads (one category, one region…) also work, but journals outside your filter will show as “unranked”.
